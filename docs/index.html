Trabalho de Grupo - Manipulação de Vetores

Introdução

Este projeto é desenvolvido no âmbito da unidade curricular de Laboratórios de Programação (LP) e consiste na implementação de um programa em linguagem C que permite realizar cálculos e operações estatísticas sobre um vetor de 14 números inteiros.

O objetivo é solicitar ao utilizador 14 números inteiros, garantindo que estão no intervalo [3, 31] através da validação de entrada. Após a leitura, o programa apresenta um menu interativo que permite aceder a diversas funcionalidades.

Funcionalidades Mínimas

As funcionalidades básicas implementadas incluem:

  • Cálculo da raiz quadrada de todos os elementos.
  • Cálculo da média de todos os elementos.
  • Devolução dos valores do vetor divisíveis por três.
  • Devolução do vetor ordenado por ordem decrescente.
  • Construção de uma matriz 14x14 baseada no vetor lido e nas permutações.
  • Identificação do valor mais próximo de 25.

Funcionalidades Adicionais

A versão mais elaborada do projeto inclui adicionalmente:

  • Leitura de um novo vetor e cálculo do produto interno.
  • Identificação de todos os números compostos (não primos) no vetor inicial.
  • Leitura de um novo vetor e cálculo da matriz resultante do produto do vetor inicial.
  • Cálculo do determinante da matriz resultante do produto (se aplicável).
  • Adição de uma página de ajuda no menu.
  • Ajuda na linha de comandos com a flag –help.

Estrutura do Projeto

O projeto está organizado nos seguintes ficheiros:

  • main-team-7.c — Ficheiro principal com a função main
  • functions-team-7.c — Implementação das funções
  • functions-team-7.h — Documentação das funções
